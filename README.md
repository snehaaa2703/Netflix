# Netflix Landing Page Clone 🎬

## Project Overview
This is a simple Netflix landing page clone that I created using HTML and CSS.  
I built this project to practice my frontend development skills and understand how a real website landing page is designed.

While working on this project, I tried to recreate the layout of the Netflix homepage including the hero section, feature sections, FAQ section, and footer.

This project helped me improve my understanding of HTML structure and CSS styling.

---

## Technologies Used
- HTML5
- CSS3

---

## Features
- Netflix style landing page UI
- Hero section with email input
- Feature sections explaining platform benefits
- FAQ section layout
- Footer with multiple links

## How to Run the Project

1. Download or clone this repository.

2. Open the project folder.

3. Open **index.html** in any web browser.

The project will run locally on your system.
